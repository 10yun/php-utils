
# 数组


- 功能

| 方法         | 说明                   | 详情 |
| :----------- | :--------------------- | :--- |
| columnsNeed  | 数组筛选自己想要的字段 |      |
| columnsDel   | 二维数组删除不要的字段 |      |
| multiArrSort | 多维数组排序           |      |
| treeEn       | 一维转树形             |      |
| inExistStr   | 多维数组是否存在某个值 |      |
| addToMerge   | 追加                   |      |



- 使用

```php

use shiyunUtils\helper\HelperArr;

HelperArr::方法();
```
