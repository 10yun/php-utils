# 数字

- 功能

| 方法      | 说明      | 详情 |
| :-------- | :-------- | :--- |
| arrToXml  | 数组转xml |      |
| xmlToArr  | xml转数组 |      |
| jsonToXml | json转xml |      |
| xmlToJson | xml转json |      |



- 使用

```php

use shiyunUtils\libs\LibXml;

LibXml::方法();

# 或者

\shiyunUtils\libs\LibXml::方法();

```
